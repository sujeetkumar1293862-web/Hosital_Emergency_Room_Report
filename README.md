# Hosital_Emergency_Room_Report

Hospital Emergency Room Dashboard Analysis
Excel Project Documentation
1. Project Purpose
The main objective of this project is to analyze the data of the Hospital Emergency Room (ER).
With the help of this dashboard, hospital management can easily monitor patient flow, waiting time, and service quality. This helps them make better and more informed decisions to improve hospital services.
________________________________________
2. Tools Used
This project was created using Microsoft Excel.
The main Excel features used in this project are:
•	Pivot Tables
•	Charts
•	Dashboard Design
________________________________________
3. Data Analysis
In this project, the hospital ER data was analyzed to generate useful insights.
The main analyses include:
1. Patient Admission Status
This analysis shows how many patients were admitted and how many were not admitted.
2. Patient Age Distribution
Patients were grouped into different age categories to better understand the age distribution.
3. Timeliness (30-Minute Rule)
This analysis measures how many patients were seen by a doctor within 30 minutes.
4. Gender Analysis
Patients were analyzed based on gender, such as male and female.
5. Department Referrals
This analysis shows which hospital departments received the most referrals from the emergency room.
________________________________________
4. Dashboard
All the analysis results are presented in an interactive Excel dashboard.
The dashboard displays important information such as:
•	Patient trends
•	Wait time analysis
•	Gender distribution
•	Department referrals
All insights are clearly shown on a single screen for easy understanding.
